前 端 深 层 技 术 介 绍

ES6

ES6 主要是为了解决 ES5 的先天不足，但是目前浏览器的 JavaScript 是 ES5 版本，大多数高版本的浏览器也支持 ES6，不过只实现了 ES6 的部分 功能。

jQuery

jQuery是一个JavaScript函数库,还提供了大量的插件。

jQuery库包含以下功能:JavaScript 特效和动画,HTML DOM 遍历和修改,AJAX和Utilities

Vue.js

Vue 只关注视图层， 采用自底向上增量开发的设计。

Vue 的目标是通过尽可能简单的 API 实现响应的数据绑定和组合的视图组件。

Bootstrap

Bootstrap，来自 Twitter，是目前最受欢迎的前端框架。

Bootstrap 是全球最受欢迎的前端组件库，用于开发响应式布局、移动设备优先的 WEB 项目。

AJAX

AJAX 不是新的编程语言，而是一种使用现有标准的新方法。

AJAX 最大的优点是在不重载整个页面的情况下，可以与服务器交换数据并更新部分网页内容。

Node.js

简单的说 Node.js 就是运行在服务端的 JavaScript。

Node.js是一个事件驱动I/O服务端JavaScript环境，基于Google的V8引擎,性能非常好。
